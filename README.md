# Test Cat Pragma
Aplicación Catbreeds

## Instrucciones de operación

1. Configurar el entorno de desarrollo de Flutter.

2. Clone el código, ejecute 'flutter packages get' para instalar paquetes de terceros.
```
flutter packages get
```
